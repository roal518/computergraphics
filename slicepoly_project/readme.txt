readme 

도형은 여러번 슬라이스 할 수 있습니다.
3차 매개변수 곡선을 사용해서 도형의 경로를 표현하기는 생략했습니다.
확인하기 쉽도록 슬라이스 하면 도형의 색상이 변경됩니다.



keys
'1' 번키 line/fill 결정 line 모드로 보면 더 직관적입니다.

-/+ 키(쉬프트 상관 X):도형의 이동속도를 조절합니다.
-:감속
+:가속

[/] 키 도형의 생성 속도를 조절합니다.
[: 빠르게 생성
]: 느리게 생성

</> 키(쉬프트키 상관 X): 도형의 모양 범위가 조절됩니다. 기본(3~5각형)에서 감소하지 않습니다.
<:3~n-1각형이 됩니다.
>:3~n+1각형이 됩니다. //쭉 누르면 확인하기 편합니다.

**주의**
무한에 가까운 다각형을 그릴수 있지만 생성 속도가(이동속도 X) 최대 속도에 근접하게 되면 약간 느려질수 있습니다.
이때,이동속도를 빠르게 해주시면 원활하게 보실 수 있습니다.
